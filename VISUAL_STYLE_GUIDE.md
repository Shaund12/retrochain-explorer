## Animation Details

### Toast Animations
1. **Entrance**: Slide from right + bounce
2. **Progress Bar**: Linear countdown
3. **Icon**: Static (no animation)
4. **Exit**: Fade out + slide right
5. **Drag**: User can drag to dismiss

### Disclaimer Animations
1. **Entrance**: Fade in + slide down
2. **Background**: Pulsing ambient glow
3. **Icon**: Pulse animation (2s infinite)
4. **Exit**: Fade out (if dismissible)

## Responsive Behavior

### Desktop (1024px+)
```

- Position: top-right, 24px from edge
- Width: 400px max
- Stack vertically

Disclaimer:
- Full width of content area
- Two-column layout (icon + content)
```

### Tablet (768px - 1023px)
```
Toast:
- Position: top-right, 16px from edge
- Width: 360px max
- Stack vertically

Disclaimer:
- Full width of content area
- Two-column layout (icon + content)
```

### Mobile (<768px)
```
Toast:
- Position: top-center, 12px from edge
- Width: calc(100vw - 32px)

Disclaimer:

- Single column layout
- Icon above content
```
